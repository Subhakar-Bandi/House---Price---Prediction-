# 🏠 House Price Prediction
This project predicts house prices using Linear Regression in Python.
## 🔧 Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
## 📂 Dataset
Contains columns like:
- square_feet
- num_rooms
- age
- distance_to_city(km)
- price
## 📈 Model Used
- Linear Regression
## 💡 Accuracy
R² Score: ~0.85
## 📦 How to Run
1. Upload the dataset in Colab
2. Run the notebook cells
3. Predict new house price easily
